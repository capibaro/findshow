# FINDSHOW

[FindShow](https://www.findshow.xyz/) 是一个独立音乐演出搜索网站，它通过 [ShowSpider](https://github.com/capibaro/ShowSpider) 收集 LiveHouse 和音乐节的演出信息，可以根据音乐人、场馆和城市搜索你感兴趣的演出。
